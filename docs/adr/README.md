# Architecture Decision Records (ADR)

This folder stores architecture decisions that affect security, reliability, and platform behavior.

## Naming and Format

- File naming: `NNNN-short-title.md` (example: `0001-production-hardening-baseline.md`)
- Status values: `Proposed`, `Accepted`, `Superseded`
- Required sections:
  - `Status`
  - `Context`
  - `Decision`
  - `Consequences`
  - `Verification`

## Index

- `0001-production-hardening-baseline.md` - Baseline controls for connector security, validation, and governance.
