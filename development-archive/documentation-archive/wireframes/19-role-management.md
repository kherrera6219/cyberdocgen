## Interactions

### 1. Create Custom Role
1. User clicks "+ Create Custom Role"
2. Opens multi-step modal
3. Enters role name and description
4. Optionally selects base role
5. Selects individual permissions
6. Reviews and confirms
7. System creates role

### 2. Edit Role Permissions
1. User clicks "Edit Role"
2. Opens permission editor
3. Checks/unchecks permissions
4. Clicks "Save Changes"
5. System validates and updates
6. Affected users see new permissions

### 3. View Permission Matrix
1. User clicks "Permission Matrix" tab
2. Shows comparison table
3. Can filter by permission category
4. Can export to CSV

### 4. Duplicate Role
1. User clicks "Duplicate"
2. Creates copy with "_copy" suffix
3. Opens for editing
4. User modifies as needed
5. Saves new role

### 5. Delete Custom Role
1. User clicks "Delete"
2. Shows confirmation if users assigned
3. User selects reassignment role
4. Confirms deletion
5. System reassigns users
6. Deletes role

### Permission Checking
```typescript
// Check if user has permission
function hasPermission(user: User, permissionKey: string): boolean {
  const userRole = user.role;
  const permissions = userRole.permissions;

  return permissions.some(p => p.key === permissionKey);
}

// Middleware for route protection
function requirePermission(permissionKey: string) {
  return (req, res, next) => {
    if (!hasPermission(req.user, permissionKey)) {
      return res.status(403).json({
        error: 'Insufficient permissions',
      });
    }
    next();
  };
}

// Usage in routes
router.post('/documents',
  requirePermission('documents.create'),
  createDocument
);
```

### Role Updates
```typescript
async function updateRole(roleId: string, updates: RoleUpdate) {
  // Check if system role
  const role = await Role.findById(roleId);
  if (role.type === 'system') {
    throw new Error('Cannot modify system roles');
  }

  // Update role
  await Role.update(roleId, updates);

  // Refresh permissions for all users with this role
  const users = await User.find({ roleId });
  for (const user of users) {
    await refreshUserPermissions(user.id);
  }

  // Invalidate permission cache
  cache.invalidate(`role:${roleId}:permissions`);

  return role;
}
```
