## Key Components

### Header Actions
- **New Document:** Primary button, opens document creation modal
- **Export:** Secondary button, exports selected documents

### Search & Filters Bar
- **Search:** Full-text search across document names/content
- **Filter:** Dropdown with:
  - Framework (ISO 27001, SOC 2, etc.)
  - Status (Draft, Review, Approved, Archived)
  - Date range
  - Author
- **Sort:** Dropdown with:
  - Name (A-Z, Z-A)
  - Date (Newest, Oldest)
  - Status
- **View:** Toggle between List / Grid / Compact views

### Category Tabs
- All documents
- Filter by document type
- Shows count for each category
- Horizontally scrollable on mobile

### Documents Table
- **Checkbox:** Bulk select documents
- **Name:** Document title with icon
- **Framework:** Associated compliance framework(s)
- **Status:** Visual badge with color
- **Updated:** Relative timestamp
- **Actions:** Kebab menu (â‹®) with:
  - View
  - Edit
  - Download
  - Share
  - Archive
  - Delete
