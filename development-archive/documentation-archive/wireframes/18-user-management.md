## Interactions

### 1. Invite User
1. User clicks "+ Invite User"
2. Opens invite modal
3. Enters user details
4. Selects role
5. Configures options
6. Clicks "Send Invitation"
7. System sends email
8. User appears in pending list

### 2. Edit User
1. User clicks "Edit" on user row
2. Opens edit modal
3. Modifies user details
4. Changes role if needed
5. Clicks "Save Changes"
6. System updates user
7. Shows success message

### 3. Suspend User
1. User clicks "Suspend"
2. Shows confirmation dialog
3. User enters reason
4. Clicks "Suspend User"
5. System immediately revokes access
6. Invalidates active sessions
7. User status changes to suspended

### 4. Bulk Actions
1. User selects multiple users via checkboxes
2. Selects bulk action from dropdown
3. Confirms action
4. System processes all selected users
5. Shows progress and results

### 5. Search & Filter
1. User enters search query
2. System filters results in real-time
3. Can filter by role, status, department
4. Can sort by name, email, last active
5. Pagination updates automatically

## User Roles & Permissions

### Viewer
- View documents
- View compliance status
- View reports (read-only)
- Cannot edit or create

### Editor
- All Viewer permissions
- Create/edit documents
- Manage own content
- Generate reports
- Cannot manage users or settings

### Auditor
- All Viewer permissions
- Access audit trails
- Generate audit reports
- View all activities
- Cannot edit content

### Admin
- All permissions
- Manage users
- Configure settings
- Access all features
- Manage billing

### Bulk Import Processing
```typescript
async function processBulkImport(file: File, options: ImportOptions) {
  const users = await parseCSV(file);
  const results = {
    successful: [],
    failed: [],
  };

  for (const userData of users) {
    try {
      // Validate email
      if (!isValidEmail(userData.email)) {
        throw new Error('Invalid email');
      }

      // Check for duplicates
      const exists = await userExists(userData.email);
      if (exists) {
        throw new Error('User already exists');
      }

      // Create invitation
      const invitation = await createInvitation({
        email: userData.email,
        firstName: userData.firstName,
        lastName: userData.lastName,
        role: userData.role || 'viewer',
        department: userData.department,
      });

      // Send email if enabled
      if (options.sendEmails) {
        await sendInvitationEmail(invitation);
      }

      results.successful.push(userData.email);
    } catch (error) {
      results.failed.push({
        email: userData.email,
        error: error.message,
      });
    }
  }

  return results;
}
```
